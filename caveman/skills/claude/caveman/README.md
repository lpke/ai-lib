# Caveman Claude Skill

Claude skill package for compressed technical communication.

## Install

Personal:

```sh
mkdir -p ~/.claude/skills
ln -s "$PWD/caveman/skills/claude/caveman" ~/.claude/skills/caveman
```

Project:

```sh
mkdir -p .claude/skills
ln -s "$PWD/caveman/skills/claude/caveman" .claude/skills/caveman
```

The package must contain `SKILL.md` at the skill root.
